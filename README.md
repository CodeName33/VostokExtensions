# Описание
## Модели
 - **Вывод LAN (нужно купить переходник).stl**
 - **Держатель подсветки.stl** - Для светодиодной ленты, ставится между катушек, под турбиной (пока это промежутоный вариант, мне не очень нравится, как освещает)
 - **Для камеры.stl** - заглушка, в которуя я вставил камеру (делается отверстие в стенке и вставляется заглушка), тоже не лучший вариант, но как-то работает, возможно еще буду менять
 - **Переделаные чистилки и формы.stl** - Немного другие кревления под чистилки, где сопло будет заткнуто в припаркованм положении. Также формы для создания "силиконовой" части из автомобильного красного керметия (используется для прокладок коробок, двигателся, держит 400 градусов)
 - **Планка для закрепления стоек оси Z.stl** - В 9й версии принтера там заглушка, которая никак не крепится, я сделал такую, крепится в отверстия в зашивке, идет поверх заглушки, уменьшаяет область печати по Z на пару мм.
 - **Площадка нажимать концевик.stl** - там болтик на левой голове, которые нажимает концевик на правой, он не всегда попадает в него. Этот переходнрик расширяет область нажатия
 - **Шаблоны для сверления.stl** - Шаблоны для сверления профилей
